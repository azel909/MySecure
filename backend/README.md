# MySecure Government Audit Backend

This backend stores Malaysian government Android APK compliance and forensic audit reports generated by the Flutter app.

It does **not** require APK uploads. Store government audit metadata first: package name, risk score, MAC/PDPA findings, APK SHA-256, install/update timestamps, and forensic evidence fields.

## Main Endpoints

- `GET /health`
- `POST /auth/register`
- `POST /auth/login`
- `GET /auth/me`
- `POST /auth/logout`
- `POST /reports` requires bearer token
- `GET /reports` requires bearer token
- `GET /reports/{id}` requires bearer token
- `DELETE /reports/{id}` requires bearer token
- `GET /stats` requires bearer token

## Authentication

The first registered user becomes `admin`. Later registered users become `examiner`.

Register:

```powershell
$body = @{
  name = "Ridzuan"
  email = "ridzuan@example.com"
  password = "password123"
} | ConvertTo-Json

$auth = Invoke-RestMethod -Method Post -Uri http://localhost:8000/auth/register -ContentType "application/json" -Body $body
$token = $auth.accessToken
```

Login:

```powershell
$body = @{
  email = "ridzuan@example.com"
  password = "password123"
} | ConvertTo-Json

$auth = Invoke-RestMethod -Method Post -Uri http://localhost:8000/auth/login -ContentType "application/json" -Body $body
$token = $auth.accessToken
```

Use token:

```powershell
$headers = @{ Authorization = "Bearer $token" }
Invoke-RestMethod -Uri http://localhost:8000/auth/me -Headers $headers
```
